# Smart Chart V2 - Chat with Documents & Web

## Project Overview
Smart Chart V2 is an advanced AI-powered document and web content querying system that enables users to upload various types of documents and website links, then engage in natural language conversations to extract information and insights from their content. The system uses cutting-edge Retrieval-Augmented Generation (RAG) technology to provide accurate, context-aware responses based solely on the uploaded materials.

## Core Functionality

### Document Processing & Upload
- **Supported File Types**: PDF, DOCX, TXT, images (PNG/JPG/JPEG), Markdown, PowerPoint (PPTX/PPT), HTML files
- **File Size Limit**: 10MB per file
- **Document Chunking**: Intelligent text splitting with configurable chunk size (1000 words) and overlap (200 words)
- **Content Extraction**: Advanced text extraction including OCR for images, table parsing, and structured content handling

### Web Scraping & Processing
- **Website Link Processing**: Extract clean, readable content from web pages
- **Content Filtering**: Removes navigation, headers, footers, and scripts for clean text extraction
- **URL Validation**: Automatic URL cleaning and validation
- **Content Limits**: 10MB webpage size limit with timeout handling

### AI-Powered Chat Interface
- **Natural Language Queries**: Ask questions in plain English about uploaded content
- **Context-Aware Responses**: Answers based only on provided documents and websites
- **Source Citation**: Every response includes source references with confidence scores
- **Confidence Scoring**: Numerical confidence ratings for response reliability
- **Session Management**: Persistent context across conversations

### Technical Architecture

#### Backend (FastAPI)
- **Framework**: FastAPI with async support for high performance
- **API Endpoints**:
  - `/upload`: Document file upload and processing
  - `/process-link`: Website content scraping and indexing
  - `/query`: Natural language question answering
  - `/clear-context`: Remove all indexed content
  - `/context-stats`: System status and statistics
  - `/health`: Health check endpoint

#### AI & Machine Learning Components
- **Embedding Model**: sentence-transformers/all-MiniLM-L6-v2 (384-dimensional vectors)
- **Vector Database**: FAISS (Facebook AI Similarity Search) for efficient similarity search
- **LLM Integration**: Together AI API with Meta Llama 3.3 70B Instruct model
- **RAG Implementation**: Custom retrieval system combining semantic search with generative AI

#### Frontend (HTML/JavaScript)
- **UI Framework**: Bootstrap 5 with custom styling
- **Real-time Interface**: Dynamic chat interface with message history
- **File Upload**: Drag-and-drop support with progress indicators
- **Status Dashboard**: Real-time system statistics and document counts
- **Responsive Design**: Mobile-friendly interface

#### Data Processing Pipeline
1. **Document Upload**: File validation and temporary storage
2. **Text Extraction**: Format-specific processing (PDF OCR, DOCX parsing, etc.)
3. **Chunking**: Intelligent text segmentation with overlap
4. **Embedding Generation**: Convert text chunks to vector representations
5. **Indexing**: Store vectors in FAISS index with metadata
6. **Query Processing**: Semantic search for relevant chunks
7. **Context Assembly**: Combine relevant chunks with source information
8. **AI Generation**: Generate responses using retrieved context
