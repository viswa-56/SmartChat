**AI-Powered Chat with Documents & Web**  
Smart Chart V2 (enhanced SmartChat): Upload documents (PDF, DOCX, images, PPTX, etc.), process websites, ask questions. Accurate RAG answers with citations/confidence—FAISS + Llama 3.3 70B.

## ✨ Features
- Multi-Format Support: PDF, DOCX, TXT, MD, PPTX, HTML, images (OCR)
- Web Scraping: Clean URL extraction (trafilatura)
- RAG: Semantic search + LLM
- Confidence scores & sources
- Real-time stats dashboard
- Dark Bootstrap UI
- Session clear

## 📋 Supported Formats
| Type | Examples | Processing |
|------|----------|------------|
| Docs | PDF, DOCX | Extraction + chunking |
| Slides | PPTX | Notes/slide parse |
| Web | URLs | Article clean |
| Images | PNG/JPG | OCR |

## 🚀 Quick Start
1. `pip install -r requirements.txt`
2. `.env`: `HUGGINGFACE_API_KEY=your_key` (Together AI)
3. `python app/main.py`
4. http://localhost:8000
